# コピーライティング原則(ダメ vs 良い)

Threads向け hook + body の品質基準。
**hook と body は役割が完全に違う**。混同しない。

---

## 大原則

### hook の役割
**指を止めること**。それだけ。

- 逆説的な観察(1〜2文)
- 痛みの瞬間 or セリフ(1〜2文)

これ以外を入れない。**人物名・著作名・学者名・専門用語は hookに入れない**。

### body の役割
**真実を見せること**。ここで初めて、心理学者・研究・概念を**砕いた言葉で**出す。

---

## 1. 「hook」 — ダメ vs 良い

### ❌ ダメ: 人物名+著作名で始まる

```
ピーター・ドラッカーが『マネジメント』で喝破した、30代で起業した一人社長ほど陥る「集客の罠」。
なぜ「良いサービスを作れる」と誇りに思っていても、月末、銀行口座を見る前の一呼吸で
「また今月もフォロワーが増えない…」と失敗動詞してしまうのか。
```

- 人物名・著作名が hook に出てる → 仰々しい・難しい
- 「失敗動詞」って何? → 不自然な日本語(プレースホルダの残骸)
- 1文が長すぎる

### ❌ ダメ: 専門用語で始まる

```
「メンタルブロック」という言葉、知ってますよね。
でも、それを「自分を守る悪いもの」だと思っている人ほど、
「指導教員に相談するの怖い…」という痛みの瞬間にハマる。
```

- 「メンタルブロック」が業界用語のまま出ている
- 業界外の人には意味不明
- 「ハマる」とか「罠」とか定型化

### ❌ ダメ: ポエム・スピ調

```
一日を終え、今日の自分を労うように、そっと目を閉じるあなたへ。
「与えること」は得意なのに「受け取れない」という悩みは、
あなたが『すでに満たされている』という内なる豊かさを認識することで…
```

- スクロール中の人の指は止まらない
- 抽象概念のオンパレード

### ✅ 良い: 逆説 + 痛みの瞬間 だけ

```
クライアントを癒やすほど、自分のセッション料を下げてしまう。
「今回は特別に半額で…」と、また言ってしまった。
```

- 1文が短い(40字以内)
- 業界の人にも一般人にも分かる日常語
- 一般常識に反する観察(癒やせる人=料金取れる人、ではない)
- ターゲットが本当に発するセリフ
- 人物名・専門用語・固有名詞ゼロ

## 2. 「body」 — ダメ vs 良い

### ❌ ダメ: 概念の上塗り

```
内なる豊かさを認識し、それを受け取るに値すると信じる『心の状態』を整える。
```

- 抽象 + 抽象 + 抽象 = 何も残らない

### ❌ ダメ: 学術調そのまま

```
アダム・グラントが提唱したギブ&テイク理論によれば、
成功するギバーは戦略的アプローチを採用することで…
```

- 「提唱した」「採用することで」が固い
- 4歳の子に説明する語り口になってない

### ✅ 良い: 否定→砕いた権威→ターゲットの場面→寄り添い

```
それ、性格でも能力の問題でもない。

心理学者のアダム・グラントってアメリカの学者がね、
「与えすぎる人」を何年も追跡してて、面白いことを言ってる。
こういう人ほど、自分の価値を低く見積もる傾向があるって。

ハーバードの追跡調査でも、援助職の人にこのパターンが
特に出やすいことが分かってる。

つまり、クライアントに「お願いします」と言われた瞬間に
喉が締まって「今回は特別に半額で…」と口走るのは、
何年もかけて身につけた自分を守る癖、ってこと。

それは弱さじゃない。
気づいた今、何が変わると思う?
```

良い点:
- 「学者がね」「面白いことを言ってる」と話し言葉
- 学者名(アダム・グラント)も研究機関(ハーバード)も出てるが、**砕いている**
- ターゲットの具体的場面(「お願いします」と言われた瞬間)に接続
- 締めは指示じゃなく問いかけ

## 3. body の組み立て手順(順番厳守)

### Step 1: 否定する
読者が思い込んでいる「ありがちな原因」を1行で否定する。
```
それ、性格の問題じゃない。
能力が低いからじゃない。
努力が足りないからじゃない。
```

### Step 2: 砕いた言葉で権威を出す
人名・研究・概念を**ここで初めて**出す。ただし話し言葉で砕く。

✅ 良い砕き方:
- 「心理学者の○○ってアメリカの学者がね。」
- 「○○大学のチームが□年追ってる研究があって。」
- 「行動経済学に『○○』って考え方があるんだけど、簡単に言うと△△ってこと。」

❌ 砕いていない:
- 「○○博士が提唱した△△理論によると」
- 「□□論文で明らかになった」

### Step 3: ターゲットの今日の場面と紐づける
権威で語ったことを、ターゲットの**今日の1日のある場面**に降ろす。
```
これって、あなたが○○の瞬間に△△と感じる、あの感覚と同じ。
```

### Step 4: 寄り添いで締める
指示形は禁止。問いかけ or 観察で終わる。
```
それは弱さじゃない。
気づいた今、何が変わると思う?
```

## 4. 禁止フレーズ集(逐語)

過去の生成で失敗した実例。これらが含まれていたら即ボツ。

### 抽象概念オンパレード(NG)
- 内なる豊かさ / 内なる安全性 / 自己価値
- 心の状態(を整える)
- 受け取り体質 / 受け取り上手 / 与える練習 / 健全な循環
- 真のギバー / 偽りの自己

### 教科書まとめ調(NG)
- ○○することで解決できます
- ○○を整えることで
- ○○することが大切です
- ○○を認識し、~と信じる
- 必要があります / 構いません

### 予言・スピ調(NG)
- ここから目覚めます
- 本来のあなたが
- 内側から変わります
- 引き寄せられます

### 指示形(NG)
- ○○してみませんか?
- ○○してみてください
- ○○しましょう
- 試してみませんか?

### 詩・ポエム導入(NG)
- そっと目を閉じるあなたへ
- 一日を終え、今日の自分を労うように
- 静かに耳を傾けてみてください

### 権威の偽装(NG)
- 海外の心理学者によると
- ある研究では
- 専門家が言うように
- 多くの研究が示すように

### 変数の残骸(NG・絶対に出すな)
- 失敗動詞 / 失敗動作 / 行動動詞
- [○○] のような角括弧つき表記
- 「失敗してしまう」「失敗動詞してしまうのか」など不自然な動詞

### hook限定NG(★最重要★)
- **人物の固有名**(アダム・グラント / アリス・ミラー / ドラッカー / 神田昌典 等)
- **著作名**(『マネジメント』『才能ある子どものドラマ』等)
- **学者名・研究機関名**(ハーバード大 / Stanford 等)
- **専門用語**(メンタルブロック / DRM / ファネル / 防衛機制 / メイラード反応 等)

これらは **body の中** で砕いて出す。hookには絶対に出さない。

---

## 5. まとめ

| 場所 | やる | やらない |
|---|---|---|
| **hook** | 逆説 + 痛みの瞬間 + セリフ | 人物名 / 著作名 / 学者名 / 専門用語 |
| **body** | 砕いた権威(○○ってアメリカの学者がね…)+ ターゲットの今日の場面 + 寄り添い | 学術調 / 抽象概念 / 教科書まとめ調 |
