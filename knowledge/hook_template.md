# フック設計テンプレート(汎用・全分野共通)

## 大原則(超重要)

### hook の役割
**指を止める** こと。それ以上のことをしない。
権威の引用は **bodyの中** で行い、hookには絶対に出さない。

### hook の構造(全分野共通・1つだけ)
hook は以下2要素のみで構成する。これ以外を入れない:

1. **逆説的な観察**(1〜2文)
   → 一般常識・業界常識をひっくり返す観察
2. **痛みの瞬間 or セリフ**(1〜2文)
   → ターゲットが実際に体験する具体的な瞬間 + 口にする/頭でつぶやくセリフ

文字数目安: hook 全体で **60〜120字**。
1文40字以内を守る。改行を活かす。

---

## hook で絶対にやらないこと(即ボツ)

- **人物の固有名を入れない**(アダム・グラント / アリス・ミラー / ドラッカー / 神田昌典 等 全部NG)
- **著作名を入れない**(『マネジメント』『才能ある子どものドラマ』『ニュー・アース』等 全部NG)
- **学者名・研究機関名を入れない**(ハーバード大 / スタンフォード心理学部 / 等 全部NG)
- **専門用語を出さない**(メンタルブロック / DRM / ファネル / 防衛機制 / メイラード反応 / IFS / アタッチメント / 等)
- **詩・ポエム導入**(「そっと目を閉じるあなたへ」)
- **教科書まとめ調**(「○○することで解決できます」「○○することが大切です」)
- **指示形**(「○○してみませんか?」「○○してみてください」「○○しましょう」)
- **予言・スピ調**(「目覚めます」「内側から変わります」)
- **抽象概念オンパレード**(「内なる豊かさ」「心の状態」「自己価値」)
- **変数の取りこぼし**(「失敗動詞」「失敗してしまう」みたいな自然じゃない動詞は使わない)

## hook の良い例(各分野)

### 例1: ヒーリング・コーチング
```
クライアントを癒やすほど、自分のセッション料を下げてしまう。
「今回は特別に半額で…」と、また言ってしまった。
```

### 例2: ビジネス・経営
```
良いサービスを作ったのに、売れない。
月末、口座を見るのが怖い。
```

### 例3: スピリチュアル
```
ヨガを10年。瞑想も毎日。
なのに、夜は眠れない。
```

### 例4: 心理学・カウンセリング
```
セッション中は集中できるのに、終わった瞬間に頭が回り出す。
「あの介入であってたかな」と、夜中まで反芻してしまう。
```

### 例5: 育児・子育て
```
仕事は段取り上手なのに、朝の玄関で崩れる。
「もう知らない!」と、まだ4歳の息子に怒鳴ってしまう。
```

### 例6: 料理・食
```
レシピ通りに作ってるのに、味が決まらない。
「やっぱりセンスないのかも」と、料理が億劫になってきた。
```

→ どれも **人名なし・専門用語なし・短い・逆説 + 痛みの瞬間** だけで構成されている。
→ 60〜100字に収まっている。1文も短い。

## body の役割(ここに権威を置く)

hook で人を止めたら、body で**真実を見せる**。
ここで初めて、心理学者名・実績・研究データ・概念を**砕いた言葉で**出す。

### body の組み立て手順(順番厳守)

#### Step 1: 否定する
読者が思い込んでいる「ありがちな原因」を1行で否定する。
```
それ、性格の問題じゃない。
能力が低いからじゃない。
努力が足りないからじゃない。
```

#### Step 2: 砕いた言葉で権威を出す
ここで初めて、人名・研究・概念を出す。ただし**4歳の子に説明するように**砕く。

✅ 良い砕き方の例:
- 「心理学者の○○ってアメリカの学者がいてね。」
- 「○○大学のチームが□年追ってる研究があって。」
- 「行動経済学に『○○』って考え方があるんだけど、簡単に言うと△△ってこと。」

❌ 砕いていない例:
- 「○○博士が提唱した△△理論によると」(肩書き+理論名は固い)
- 「□□論文で明らかになった」(学術用語のまま)

#### Step 3: ターゲットの場面と紐づける
権威で語ったことを、ターゲットの**今日の1日のある場面**に降ろす。
```
これって、あなたがクライアントに「お願いします」と言われた瞬間に
心臓がドキッとして「今回は特別に…」と口走るあの感覚と同じ。
```

#### Step 4: 寄り添いで締める
指示形は禁止。「○○しよう」「○○してみて」NG。
代わりに**問いかけ** or **観察**で終わる。
```
それは弱さじゃない。何年もかけて身につけた、自分を守る癖。
気づいた今、何が変わると思う?
```

## body の良い例(ヒーラー分野・ヒント用)

```
それ、性格でも能力の問題でもない。

心理学者のアダム・グラントってアメリカの学者がね、
「与えすぎる人」を何年も追跡してて、面白いことを言ってる。
こういう人ほど、自分の価値を低く見積もる傾向があるって。

ハーバードの追跡調査でも、援助職の人にこのパターンが
特に出やすいことが分かってる。

つまり、あなたがクライアントに「お願いします」と言われた瞬間に
喉が締まって「今回は特別に半額で…」と口走るのは、
何年もかけて身につけた自分を守る癖、ってこと。

それは弱さじゃない。
気づいた今、何が変わると思う?
```

→ Step 1〜4 の流れに沿っている。
→ 人名(アダム・グラント)も研究(ハーバード)も出てるが、**砕いている**。
→ 「学者がね」「面白いことを言ってる」と話し言葉。

## 5投稿(朝/午前/昼/午後/夜) の流れと hook

各投稿は独立して読まれるので、5投稿とも上記の **hook の構造**(逆説 + 痛みの瞬間)を守る。
スロットごとに角度を変える:

- **朝**: ターゲットの「今日の1日のリアルな瞬間」から入る
- **午前**: 「あなたのせいじゃない」を柔らかく言う逆説
- **昼**: 新事実(常識の逆)を**驚くほど短い1文**で言い切る
- **午後**: 別の場面で再提示(同じ気づきを違う瞬間に置く)
- **夜**: 1日の終わりの問いかけ or 小さな1歩

どのスロットでも **hookに人物名・著作名・学者名を入れない**。body で砕いて出す。
