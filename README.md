# Threads 5投稿ライター

**ターゲット(何に悩んでいる、どんな人?)を入れるだけで、その人向けの Threads 5投稿を一括生成する Streamlit ツール。**

朝/午前/昼/午後/夜の5投稿を、1日のストーリーで貫く設計(コピーライティングの **Rule of One** 原則)。

各投稿には「**何を意識してこの文を書いたか**」を一般コピーライティング観点で詳しく解説。さらに **5問のコピーライティングクイズ** が出題され、使うほど学べる設計です。

全て **Gemini 2.5 Flash の無料枠** で動作。BYO-key(各ユーザーが自分のAPIキーを入力)。

---

## 特徴

- 🎯 **ターゲット中心**: 「何に悩んでいる、どんな人?」を入力すると、その人向けの5投稿を生成
- 🧵 **5投稿の流れ**: 朝(Hook)→午前(Ask)→昼(Core)→午後(Echo)→夜(Key)。1日のストーリーで読者を引き込む
- 📐 **Rule of One**: 5投稿全体で「1ターゲット/1アイデア/1感情/1ミステリー/1アクション」を貫く
- 💡 **何を意識して書いたかを学べる**: 各投稿の `design_reason` に PASONA / PASTOR / PASBECONA / 4U / WIIFM などの一般コピーライティング原則を踏まえた解説を表示
- 📚 **コピーライティングクイズ(5問)**: 初級/中級/上級。固定3問+AI生成2問。
  - 用語問題: **LP / CTA / Hook / DRM / ONE HACK モデル**
  - 一般原則: **PASONA / PASTOR / PASBECONA / AIDA / 4U / WIIFM / Rule of One / 具体性 / ベネフィット vs 機能 / 感情×論理**
- 💾 **プロフィール保存**: 入力内容をJSONで保存→次回読み込みで一発復元
- 🔍 **リサーチ機能(任意)**: Google検索グラウンディングで権威・研究結果・概念を実際に集めて素材化(海外の一次情報を優先)
- ⚡ **無料**: Gemini 2.5 Flash の無料枠(1,500 req/日)で動作

## 使い方

### 1. APIキーの取得(無料・クレカ不要)

[Google AI Studio](https://aistudio.google.com/apikey) で Gemini APIキーを発行(Googleアカウントがあれば数分)。

### 2. 入力する情報

- **ターゲット** (必須): 何に悩んでいる、どんな人?を1人の顔が浮かぶレベルで具体的に
- **発信者プロフィール** (任意): あなたが何者で何を発信しているか
- **過去の痛み** (任意): あなた自身が乗り越えてきた経験
- **分野**: 心理学/育児/ビジネス/スピリチュアル/エッセイ など
- **トーン**: 優しい〜挑発的、柔らかく包む / グサッと言い切る
- **CTA** (任意): プロフィール誘導の文言

### 3. ローカルで動かす

```bash
git clone https://github.com/atsushi0706/threads-writer-tool.git
cd threads-writer-tool
pip install -r requirements.txt
streamlit run app.py
```

ブラウザで `http://localhost:8501` を開き、サイドバーに APIキーを貼り付けるだけで使えます。

### 4. Streamlit Cloud で公開する場合

このリポジトリを fork → [Streamlit Cloud](https://share.streamlit.io) でデプロイ。
APIキーはアプリ起動後にユーザー自身がサイドバーから入力します(サーバーには保存しません)。

## 5投稿の流れ

| 時間 | ステージ | 役割 |
|---|---|---|
| 朝 8:00 | **H** Hook | 矛盾・謎かけで指を止める |
| 午前 11:00 | **A** Ask | 「あなたのせいじゃない、構造の問題」 |
| 昼 12:00 | **C** Core | アハ体験・新事実の提示 |
| 午後 17:00 | **E** Echo | 別角度で言い直し |
| 夜 21:00 | **K** Key | 誘導(問いかけ/プロフィール/小さな行動) |

5投稿全体で **Rule of One** を貫く:
- 1ターゲット(1人の顔が浮かぶレベル)
- 1アイデア(伝える新事実は1つ)
- 1感情(揺さぶる感情は1つ)
- 1ミステリー(続きを読みたくなる謎)
- 1アクション(促す行動は1つ)

## 学習モード(コピーライティングクイズ・5問)

5投稿生成と同時に、コピーライティング学習クイズが5問出題されます。

### 出題範囲
- **用語クイズ**: LP / CTA / Hook / DRM / ONE HACK モデル
- **フレームワーク**: PASONA / PASTOR / PASBECONA / AIDA
- **原則**: 4U / WIIFM / Rule of One / 具体性 / ベネフィット vs 機能 / 感情×論理 / ヘッドライン原則 / 1人に向けて書く

### 構成
- 固定3問(quiz_pool.json から難易度別に抽選)
- AI生成2問(今回生成された5投稿に紐づけて出題)

### 難易度
- **初級**: 用語と基本概念
- **中級**: 応用判断(なぜここで具体名を入れたか、なぜこの順序か)
- **上級**: 本質(感情×論理、抽象と具体のバランス、1人向けの意味)

## API消費の目安

| 機能 | コール数 |
|---|---|
| リサーチ ON | 1コール(任意) |
| 5投稿生成 | 1コール |
| クイズ AI生成 ON | 1コール(5問のうち2問がAI生成) |

→ フル機能で **3コール / 1日**。Gemini 2.5 Flash の無料枠(1,500 req/日)で月数十日は余裕。
